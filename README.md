# Paintbrush Web App

Simple Web Paintbrush app using [Vanilla JS](http://vanilla-js.com/)

![paintbrush](https://user-images.githubusercontent.com/43776784/107315494-2d476300-6ada-11eb-9f95-e241589b68c3.gif)
<br/>

### Features
* HTML5 Canvas
* Mouse Events
* 2D Context
* 2D Painting
* Brush Size
* Image Saving
